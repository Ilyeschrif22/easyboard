I got tired of the same AI-generated, Tailwind-bloated dashboards that all look identical.
I like the shadcn aesthetic  clean, minimal, intentional  so I took that as inspiration
and built my own version from scratch.

## What this is

MYSPACE is a personal dashboard built with React, Vite, and TypeScript, styled with pure CSS
and nothing else. No utility frameworks, no component libraries, no design system dependencies.

Every pixel is written by hand.

## Why

Most dashboards today are assembled from pre-built components and AI-generated layouts.
They work, but they all end up looking the same. Building from scratch forces you to actually
understand what you are putting on the screen — every spacing decision, every color, every
layout choice is yours.

## Stack

- React
- Vite
- TypeScript
- Pure CSS

## Status

Work in progress. Built for learning, built for fun, built my way.
